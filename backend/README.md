# Backend Laravel - Skincare API

API REST Laravel pour le projet e-commerce Skincare (produits, commandes, paiement CMI).

## Démarrage rapide

```bash
# Depuis la racine du projet
cd backend

cp .env.example .env
php artisan key:generate

# Configurer la base de données dans .env puis :
php artisan migrate
php artisan serve
```

API : **http://localhost:8000**

Documentation complète : voir le `README.md` à la racine du projet.
